A complete full-stack blog application built with **MongoDB, Express.js, React.js, Node.js (MERN)** featuring user authentication, CRUD operations, image uploads, comments, and responsive UI.

https://mern-blog-plp-2025.netlify.app

## Features
- Full user authentication (Register/Login/Logout)
- Create, Read, Update, Delete blog posts
- Rich text editor with image upload
- Add comments to posts
- Category management
- Responsive design with Tailwind CSS
- Protected routes & JWT authentication
- RESTful API with proper error handling

## Tech Stack
### Frontend
- React 18 + Vite
- React Router DOM v6
- Axios
- Tailwind CSS
- Context API for auth state

### Backend
- Node.js + Express.js
- MongoDB + Mongoose
- JWT Authentication
- Multer for image uploads
- CORS & dotenv
